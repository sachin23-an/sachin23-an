## Research

**[The Complexity Trap](https://github.com/sachin23-an/the-complexity-trap)**  
*Working Paper · July 2026*

Compared six strategies (buy-and-hold to LSTM) on NIFTY 50 
over 5 years. Introduced the **Complexity Penalty** metric 
to quantify how transaction costs erode model sophistication.

Key finding: SMA break-even cost is **43x higher** than LSTM's.
At Indian market costs, LSTM is already unprofitable.
